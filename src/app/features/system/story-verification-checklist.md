# MediFlow Story Verification Checklist

- Auth: register/login/logout flows with role redirects.
- Patient: browse slots, book appointment, view appointments, cancel/reschedule constraints.
- Doctor: queue visibility, confirm/decline, consultation EMR completion flow.
- Receptionist: queue operations, check-in/no-show, schedule and exceptions UI.
- Admin: analytics dashboard, user management, CSV export trigger.
- Cross-cutting: role guards, error pages (`403`/`404`), loading/empty states.
